Python: mode options of open(filepath, mode='r')

mode=one of ‘r’ / ‘w’ / ‘x’ / ‘a’+‘+’ or nothing+‘b’ or nothing
base r open for reading (default option), seek to the beginning of the file
w open for writing, delete existing content
x open for writing, fail if file exists
a open for writing, seek to the end of the file
optional R+W + open for read+write; otherwise as above
optional binary b binary mode (instead of text mode); otherwise as above
Notes:
  • f.read() sets the read position to the end of the read content, and the write position to the end of the file.[test]
  • f.write() sets the read and write position to the end of the written content.
  • f.seek() sets the position of both f.write() and f.read().
